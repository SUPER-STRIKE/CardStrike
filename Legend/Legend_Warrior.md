## `⚔️ Executioner – Judgment`

This unit can **execute enemies**:

- If an enemy is below **15% of their Max HP**, any damage dealt by this unit will **instantly kill them**

### 🩸 Bleed Effect

Each time this unit deals damage, apply **Bleed**:

- Bleed deals **6% of target’s Max HP as damage** per turn
- Lasts for **2 turns**

If this unit attacks a different target while Bleed is active:

- The Bleed effect continues to trigger

### 🔁 Counter Effect

If this unit is attacked:

- Bleed damage is **doubled**

### 💉 Sustain

This unit heals for **50% of all Bleed damage dealt**
